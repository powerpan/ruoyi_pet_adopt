# 宠物领养平台

宠物领养平台面向有领养、送养、养宠记录和宠物服务需求的用户，提供从宠物发布、申请沟通、线下看宠、确认移交到后续回访的完整流程。平台同时保留宠物档案、健康记录、社区动态和附近服务能力，让领养前后的信息沉淀在同一个系统里。

## 项目简介

平台的核心目标是让领养过程更清晰、更可追踪。发布方可以提交待领养宠物信息，说明宠物来源、健康情况、性格特点和领养要求；申请人可以查看领养大厅、提交申请资料，并通过站内对话与发布方沟通。双方确认看宠时间和地点后，发布方再推进到移交确认，避免仅靠简单状态切换完成复杂的领养流程。

领养完成后，系统会自动生成领养人的宠物档案和回访计划。后续回访用于记录宠物健康、生活状态和异常情况，方便发布方和平台持续跟进。

## 功能概览

- 领养大厅：展示可申请、已约看宠和已领养的宠物信息，支持查看详情和提交领养申请。
- 发布领养：发布方填写宠物基础信息、健康情况、照片、性格、来源和领养要求，并等待平台审核。
- 领养申请：申请人填写真实姓名、联系方式、居住情况、养宠经验、申请理由和领养承诺。
- 申请沟通：申请人和发布方可以围绕领养条件、看宠安排和后续照护进行站内对话。
- 约看宠与移交：发布方填写看宠时间和地点后进入约看阶段，最终只能从已约看宠状态确认移交。
- 自动回访：领养确认后生成回访记录，支持填写健康情况、生活状态、图片和异常说明。
- 宠物档案：记录用户自己的宠物资料，并关联健康记录、提醒和寄养授权等日常养宠信息。
- 健康记录与提醒：维护体检、疫苗、驱虫、用药等记录，并设置后续提醒。
- 社区动态：用户可以发布养宠经验、领养故事和服务体验，形成内容交流区。
- 附近服务：用户可以查看宠物服务，提交预约咨询，与商家对话，并在服务完成后评价。
- 商家工作台：商家可维护入驻资料、服务项目、预约订单和收到的评价。
- 评价治理：商家可申请屏蔽异常评价，平台审核后决定是否隐藏，保障评价展示的可信度。
- 通知中心：领养申请、沟通消息、预约状态、评价、回访和审核结果都会进入通知提醒。
- 管理端兜底：平台可审核发布内容、商家资质、领养信息、评价屏蔽申请和异常回访。

## 业务闭环

### 领养闭环

发布宠物 -> 平台审核 -> 用户申请 -> 双方沟通 -> 安排看宠 -> 确认移交 -> 自动生成宠物档案 -> 自动生成回访计划 -> 回访归档或异常处理。

### 服务闭环

用户预约咨询 -> 商家对话确认 -> 商家接受预约 -> 服务完成 -> 用户评价 -> 商家维护评价展示 -> 平台处理评价屏蔽申请。

### 内容与通知闭环

用户发布动态或评论 -> 平台审核与治理 -> 用户互动沉淀内容 -> 关键业务节点通过通知中心提醒相关用户。
